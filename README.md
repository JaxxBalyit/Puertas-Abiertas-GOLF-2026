# Evento de Puertas Abiertas - DCI (G.O.L.F.) 🧪

### Objetivo 🚩
El evento consiste en que público en general pueda ingresar a las instalaciones de la DCI para conocer los programas de licenciatura y posgrado que se ofrecen, así como las instalaciones y recursos con los que cuenta la sede. El día se conforma de varias actividades con los distintos grupos organizados, tales como tours guiados, presentaciones de los programas, simulaciones y un sinfín de actividades más.

## Historia de GOLF 📚
El **Grupo Organizado de Licenciatura en Física** (*GOLF*) es un grupo de estudiantes de la **División de Ciencias e Ingenierías** (*DCI*) que se consolida en 2020 por iniciativa de algunos estudiantes de Licenciatura en Física, con el objetivo de promover la carrera y realizar la difusión de ciencias, principalmente en el área de la física. Actualmente el grupo cuenta con más de 20 miembros activos y ha participado en diversos eventos de difusión científica, organizando charlas, conferencias, eventos interactivos y actividades educativas para estudiantes de secundaria, preparatoria, universitarios y público en general.

## Código y Simulaciones 📊
Dentro de esta carpeta encontrarás los siguientes archivos:
- `requirements.txt`: Archivo con las dependencias necesarias para ejecutar las simulaciones.
- `README.md`: Archivo con la documentación del proyecto.
- `assets/`: Carpeta con los archivos necesarios para las simulaciones.
- `simple-pendulum.ipynb`: Jupyter Notebook con la simulación del péndulo simple.
- `sympy-for-physics.ipynb`: Jupyter Notebook con el código de la librería Sympy.

Puedes descargar el código directamente, o puedes clonarlo si prefieres. Posteriormente, te diriges a la carpeta del proyecto y puedes crear un entorno virtual para ejecutar las simulaciones.
```bash
python -m venv venv
```
Instalas las dependencias:
```bash
pip install -r requirements.txt
```
Activas el entorno virtual:
```bash
venv\Scripts\activate
```
Abres los Jupyter Notebooks y ejecutas las celdas para ver las simulaciones o el código de la librería Sympy.
